# react-native-spotlight

Walkthtought the app using nitro view wiht react native teleport

## Installation


```sh
npm install react-native-spotlight react-native-nitro-modules

> `react-native-nitro-modules` is required as this library relies on [Nitro Modules](https://nitro.margelo.com/).
```


## Usage


```js
import { SpotlightView } from "react-native-spotlight";

// ...

<SpotlightView color="tomato" />
```
